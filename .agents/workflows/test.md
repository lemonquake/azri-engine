# Browser Testing AI Shortcut

1. Check if the local development server is running in a terminal.
// turbo
2. If it is not running, start it using `npm run dev`. Wait for the port to be ready (usually `http://localhost:5173`).
3. Run the `browser_subagent` tool with the URL `http://localhost:5173`.
